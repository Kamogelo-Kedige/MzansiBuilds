# Ethical Use of AI

## How AI Was Used
AI tools were used to assist with:
- brainstorming architecture options
- refining documentation structure
- reviewing database design choices
- improving code organization ideas
- reviewing code before merges into main branch

## How AI Was Not Used
AI was not used as an unquestioned source of truth.  
All suggestions were reviewed, adapted, and validated before being included in the project.

## Developer Responsibility
Final implementation decisions, testing, security checks, and integration work remained the developer's responsibility.

## Ethical Considerations
- no sensitive secrets were shared with AI tools
- no personal production data was exposed
- generated output was checked for correctness and relevance before use
