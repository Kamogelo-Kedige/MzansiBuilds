# MzansiBuilds
A developer collaboration platform where users can create accounts, share projects they are working on, update milestones, comment on other developers' work, request collaboration, and appear on a celebration wall when projects are completed.
## Project Overview

This application was built as part of a coding challenge.  
The goal is to demonstrate software engineering competence across:

- project profiling
- version control and CI
- test-driven development
- secure-by-design thinking
- documentation
- ethical use of AI


##  Core Features

- user signup and login
- create and manage projects
- update project progress with milestones
- comment on projects
- request collaboration on projects
- celebration wall for completed projects
- live feed of developer activity
